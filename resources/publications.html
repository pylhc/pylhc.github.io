    Adding New Entries

    To add new entries, please use an abstract admonition:

    ??? abstract "Title, `Authors`, [Paper ID](Link to resource){target=_blank}"
    ```
    Bibtex entry
    ```

    Ideally, titles should be put in title case. The number of authors before resorting to et al. is left up to the committers' discretion.
