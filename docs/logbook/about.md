# LogBook Summaries

The following pages contain tables where we summarise the important entries in the logbook of the OMC activities, for different machines.

One can browse the logbooks on the [main access page][elogbook_server]{target=_blank .cern_login}, then search for "OMC" and click our logbook for either LHC activities or injector activities.
One can access the same logbooks with edit rights from within the CERN network on [this page][elogbook_server_insider]{target=_blank .cern_internal}.

!!! note "LogBook Access"
    Note that a login is required to access the logbooks.

*[OMC]: Optics Measurements and Corrections
