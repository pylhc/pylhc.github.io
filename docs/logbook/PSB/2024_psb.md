
# Timeline of PSB optics measurements in 2024

This page provides quick access links to logbook entries of relevant OMC shifts or measurements and MDs done on the PSB in 2024.

!!! info "Shift Categorization"

    The letters in the **shifts** column have the following meaning:

    - W = Weekdays (Day)
    - WN = Weekdays (Night)
    - H = Holidays or weekend (Day)
    - HN = Holidays or weekend (Night)

    Hovering the column name will show a tooltip with more information.
